Local Storage

Whats is Local Storage?

Local Storage is a web storage mechanism that allows websites to store data in a user's web browser. It provides a way for websites to save key-value pairs of data on the client side, which can be accessed and manipulated using JavaScript. Local Storage is part of the Web Storage API and is designed to store data that persists even after the browser is closed, making it useful for storing user preferences, session information, or any other data that needs to be retained across sessions.

How to Use Local Storage?

Local Storage can be used with JavaScript to store and retrieve data. You can use the `setItem()` method to store data, `getItem()` method to retrieve data, and `removeItem()` method to remove data.

Example:

Here is a simple example of how to use Local Storage to store and retrieve a user's name:

      // Store data in Local Storage
      localStorage.setItem('name', 'John Doe');

      // Retrieve data from Local Storage
      const name = localStorage.getItem('name');
      console.log(name); // Output: John Doe

      // Remove data from Local Storage
      localStorage.removeItem('name');
      // Clear all data from Local Storage
      localStorage.clear();
